# n8n Weekly GitHub Dev Summary Workflow

A complete n8n workflow that generates a weekly narrative summary of a GitHub repository's activity using Claude API, delivered via Discord webhook.

## Setup (5 steps)

### 1. Import Workflow
- Open n8n → Workflows → Import from File
- Select `weekly-dev-summary.json`

### 2. Configure Variables
In n8n Variables panel, set:

| Variable | Value | Description |
|----------|-------|-------------|
| `GITHUB_REPO` | `owner/repo` | GitHub repo to summarize |
| `ANTHROPIC_API_KEY` | `sk-ant-...` | Anthropic API key |
| `DISCORD_WEBHOOK_URL` | `https://discord.com/api/webhooks/...` | Discord webhook URL |
| `LANGUAGE` | `English` | Summary language (English/FR) |

### 3. Activate Workflow
Click **Activate** in n8n. It runs automatically every Friday at 5pm.

### 4. Test Manually
Click **Test Workflow** to trigger immediately and verify output.

### 5. Customize
- Change `LANGUAGE` to `French` for FR summaries
- Replace Discord with Slack by swapping the output node

## What it does

1. **Trigger**: Every Friday at 5pm (configurable in cron)
2. **Fetch**: GitHub API gets commits, closed issues, merged PRs from the past week
3. **Format**: Data formatted with commit messages, issue numbers, contributors
4. **Generate**: Claude Sonnet writes a narrative summary (3-4 paragraphs)
5. **Deliver**: Summary posted to Discord webhook

## Sample Output

```
📋 Weekly Dev Summary | owner/repo

This week the team closed 12 issues and merged 8 PRs...

## Key Highlights
- Major performance improvements in the API layer
- New user dashboard feature shipped
- Security patches deployed

## Looking Ahead
Next week we focus on...
---
Generated by Claude + n8n
```

## Workflow Structure

```
[Cron: Friday 5pm]
       ↓
[Get Repo Info]
       ↓
[Get Commits] [Get Issues] [Get PRs]
       ↓
[Format Data]
       ↓
[Claude API → Summary]
       ↓
[Discord Webhook]
```

## Alternative: Slack

To use Slack instead of Discord, replace the Discord node with a Slack "Message" node:
- Webhook URL: your Slack incoming webhook URL
- Message: `{{ $json.content }}`
