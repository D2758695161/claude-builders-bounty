# n8n Weekly Development Summary

> Claude Code agent workflow — generates a weekly Markdown digest of merged PRs and emails it.

**Bounty:** claude-builders-bounty#5 ($200)

## Quick Start

1. Install n8n: `npm install -g n8n`
2. Import `workflows/weekly-dev-summary/weekly-dev-summary.json`
3. Set variables: `GITHUB_TOKEN`, `GITHUB_ORG`, `GITHUB_REPO`, `SUMMARY_EMAIL`
4. Activate

Full setup: see `workflows/weekly-dev-summary/SKILL.md`
