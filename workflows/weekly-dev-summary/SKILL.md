# SKILL.md — n8n Weekly Development Summary Workflow

Automatically sends a weekly Markdown summary of all merged GitHub PRs every Monday at 9 AM.

## Setup (5 steps)

### 1. Install n8n

```bash
npm install -g n8n
# or: docker run -it --rm --name n8n -p 5678:5678 n8nio/n8n
```

### 2. Import the workflow

Open n8n UI → click "Import from File" → select `weekly-dev-summary.json`

### 3. Configure variables

Set these in n8n **Variables** (Settings → Variables):

| Variable | Value |
|----------|-------|
| `GITHUB_TOKEN` | Your GitHub PAT |
| `GITHUB_ORG` | e.g. `claude-builders-bounty` |
| `GITHUB_REPO` | e.g. `claude-builders-bounty` |
| `SUMMARY_EMAIL` | your@email.com |

### 4. Connect your email credentials

In n8n → Settings → Credentials → add an Email IMAP/SMTP credential.

### 5. Activate

Click **Activate** in the n8n workflow panel.

## What It Does

1. **Monday 9 AM** — Schedule trigger fires
2. **Fetches** all closed PRs from the past 7 days
3. **Analyzes** — counts additions/deletions, groups by label
4. **Generates** Markdown summary
5. **Sends** email to `SUMMARY_EMAIL`

## Sample Output
